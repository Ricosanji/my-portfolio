# 🚀 Nguyen Huu Cong - Portfolio

Welcome to my personal portfolio website! This is where I showcase my skills, projects, and a bit of my personality through 3D animations, slick interactions, and fluid motion.

## 🔥 Features

- **3D Animations**: Interactive keyboard using Spline with skills as keycaps that reveal titles and descriptions on hover.
- **Slick Interactions**: Powered by GSAP and Framer Motion for smooth animations on scroll, hover, and element reveal.
- **Space Theme**: Particles on a dark background to simulate a cosmic environment.
- **Music Player**: Built-in music player with audio visualizer and LED effects.
- **Responsive Design**: Fully responsive across all devices.

## 🛠️ Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS, Shadcn, Aceternity UI
- **Animations**: GSAP, Framer Motion, Spline Runtime
- **Misc**: Resend, Zod

## 🚀 Getting Started

### Prerequisites

- Node.js (v14+)
- npm

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Ricosanji/my-portfolio.git
    ```

2. Navigate to the project directory:

    ```bash
    cd my-portfolio
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Run the development server:

    ```bash
    npm run dev
    ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
